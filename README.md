Deskripsi

Proyek ini mengimplementasikan Encapsulation dan Inheritance dalam Java. 
Menampilkan pengelolaan data siswa dengan validasi,objek immutable, pewarisan kelas, polymorphism, 
serta penggunaan abstractclass untuk perhitungan area, serta manajemen list anggota tim dan proteksi data koleksi.

Enkapsulasi

1. Student1 = Mutable student dengan getter/setter dan validasi nilai.

2. Student2 = Semi-immutable student dengan metode addScore/reduceScore.

3. ImmutableStudent = Fully immutable student, perubahan score menghasilkan objek baru.

4. Team = Mengelola daftar anggota Student2 dengan proteksi encapsulation.

5. Main1, Main2, MainImmutableStudent, MainTeam = Demo penggunaan class di atas.


Inherintance

1. Account, SavingsAccount, CheckingAccount = Simulasi bank dengan inheritance dan method spesifik.

2. Vehicle, Car, Motorcycle = Contoh polymorphism melalui method start().

3. Shape, Triangle, Rectangle, Circle = Abstract class dan polymorphism, menghitung total area.

4. MainBank, MainKendaraan, MainShape3 = Demo penggunaan class di atas.
